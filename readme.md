# executable-cli [![Build Status](https://travis-ci.org/kevva/executable-cli.svg?branch=master)](https://travis-ci.org/kevva/executable-cli)

> Check if a file is executable


## Install

```
$ npm install --global executable-cli
```


## Usage

```
$ executable --help

  Usage
    $ executable <file>
```
